## Project Overview
This project focuses on building an enterprise-style healthcare analytics dashboard in Microsoft Excel using Electronic Medical Records (EMR) data.

The objective is to provide actionable insights for hospital administrators and operational teams through KPIs, trends, and interactive dashboards.

## Tools Used
- Microsoft Excel
- Power Query (Data Preparation)
- Power Pivot (Data Modeling & Measures)

## Dataset Overview
The dataset includes the following entities:
- Patients
- Doctors
- Visits
- Treatments
- Lab Results

## Project Scope (This Repository)
- Data cleaning and transformation
- Data modeling using relationships
- KPI definition and calculation
- Excel dashboard development
